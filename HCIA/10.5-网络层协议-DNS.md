定义：域名系统，本质是**分布式的域名 - IP 地址映射数据库**，核心作用是将人类易记的域名（如 `www.huawei.com`）解析为计算机可识别的 IP 地址（如 `185.199.108.153`），<font color="#ff0000">避免用户记忆复杂的 IP 地址，同时支撑域名的灵活管理</font>（如 IP 变更时无需修改用户访问方式）。

## 一、工作原理
DNS 解析是 **“从本地到全球” 的分层查询过程，核心原则是 “先缓存后查询，先本地后远程”，避免重复请求全球根服务器，提升解析效率**

查询步骤：
- 步骤 1：本地缓存查询（最快路径）
- 步骤 2：本地 DNS 服务器查询（运营商 / 自定义 DNS）
- 步骤 3：根服务器查询（全球 DNS 顶层）
- 步骤 4：顶级域（TLD）服务器查询
- 步骤 5：权威服务器查询（域名最终数据源）
- 步骤 6：结果返回与缓存

DNS 记录的缓存有效期（单位：秒），由权威服务器设置（如 300 秒 = 5 分钟、3600 秒 = 1 小时）

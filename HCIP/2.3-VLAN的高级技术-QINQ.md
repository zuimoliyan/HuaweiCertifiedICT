## 一、产生的背景
同网段互访的设备距离比较远，但是需要互访。
## 二、案例配置
### 1. 静态 QINQ 技术
核心是在核心交换机与下级的傻瓜交换机的直连接口起 `port link-type dot1q-tunnel`
按照图上配置即可一键刷命令
1. 此时 `VLAN100` 下的 `VLAN10、20` 可以访问对端 `VLAN100` 下的 `VLAN10、20`，`VLAN200` 下的同理
2. `VLAN100`和 `VLAN200`下的 `VLAN10、20` 不互通
![](DICM/Pasted%20image%2020251201104933.png)
- `SW1` 和 `SW2`
```
undo stp enable

vlan batch 100 200

interface GigabitEthernet0/0/1
 port link-type dot1q-tunnel
 port default vlan 100

interface GigabitEthernet0/0/2
 port link-type dot1q-tunnel
 port default vlan 200

interface GigabitEthernet0/0/3
 port link-type trunk
 port trunk allow-pass vlan 100 200
```
- `SW3`、`SW4`、`SW5` 、`SW6`
```
undo stp enable

vlan batch 10 20

interface GigabitEthernet0/0/1
 port link-type trunk
 port trunk allow-pass vlan 10 20

interface GigabitEthernet0/0/2
 port link-type access
 port default vlan 10

interface GigabitEthernet0/0/3
 port link-type access
 port default vlan 20
```

#### 抓包分析
![](DICM/Pasted%20image%2020251201105502.png)
此时套了两层 `802.1Q` 协议，第一层是外层，即 `VLAN100`或者 `VLAN200`，第二层是内层，即 `VLAN10`或者 `VLAN20`

### 2. 灵活的 QINQ 技术
1. **qinq vlan-translation enable** 使能VLAN转换功能
2. **port hybrid untagged vlan 2 3**  //配置Hybrid类型接口加入的VLAN，这些VLAN的帧以Untagged方式通过接口
3. **port vlan-stacking vlan 10 stack-vlan 2 remark-8021p 6**  //将 `VLAN10` 映射为 `VLAN2`，并设置优先级为6

![](DICM/Pasted%20image%2020251201113145.png)

- `SW3`和 `SW5`
```
vlan batch 10 20

interface GigabitEthernet0/0/1
 port link-type trunk
 port trunk allow-pass vlan 10 20
 
interface GigabitEthernet0/0/2
 port link-type access
 port default vlan 10

interface GigabitEthernet0/0/3
 port link-type access
 port default vlan 20
```
- `SW1` 和 `SW2`
```
vlan batch 2 3

interface GigabitEthernet0/0/1
 qinq vlan-translation enable
 port hybrid untagged vlan 2 to 3  
 port vlan-stacking vlan 10 stack-vlan 2 remark-8021p 6
 port vlan-stacking vlan 20 stack-vlan 3

interface GigabitEthernet0/0/3
 port link-type trunk
 port trunk allow-pass vlan 2 to 3
```
#### 抓包分析
![](DICM/Pasted%20image%2020251201113509.png)
`VLAN10` 下的设备发包后的优先级为6



### 3. 如果同一大 VLAN 下的两个VLAN 要互访
重点：<font color="#ff0000">在核心区（主公司的网络）配置网关</font>
![](DICM/Pasted%20image%2020251201195444.png)

`SW1和SW2` 配置
```
vlan batch 2 3

interface GigabitEthernet0/0/1
 qinq vlan-translation enable
 port hybrid untagged vlan 2 to 3  
 port vlan-stacking vlan 10 stack-vlan 2 remark-8021p 6
 port vlan-stacking vlan 20 stack-vlan 3

interface GigabitEthernet0/0/3
 port link-type trunk
 port trunk allow-pass vlan 2 to 3
```
`SW3和SW4` 配置
```
interface Vlanif10
 ip address 192.168.1.254 255.255.255.0
 
interface Vlanif20
 ip address 192.168.2.254 255.255.255.0

interface GigabitEthernet0/0/1
 port link-type trunk
 port trunk allow-pass vlan 10 20

interface GigabitEthernet0/0/2
 port link-type access
 port default vlan 10

interface GigabitEthernet0/0/3
 port link-type access
 port default vlan 20

```
